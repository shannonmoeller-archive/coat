## Install

With [Bower](http://bower.io):

    $ bower install shannonmoeller/coat

With [Component](http://component.io):

    $ component install shannonmoeller/coat

With [Node.js](http://nodejs.org):

    $ npm install coat

## Test

    $ npm test

## Compatibility

Tested to work in the following environments:

- Chrome (latest)
- Firefox (latest)
